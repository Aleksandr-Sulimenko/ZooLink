# Спецификация: Домен рынка домашних животных

## Результат
Обеспечить размещение, поиск и управление объявлениями о сопроводительных животных (домашних животных), таких как собаки, кошки, птицы и т.д. Поддержать операции продажи, разведения, усыновления и выставки. Обеспечить соответствие российским правилам торговли и рекламы животными.

## Область и границы
**Включает:**
- Объявления о домашних животных с атрибутами: название, описание, цена (или бесплатно), местоположение (гео-координаты), контактная информация (телефон, ссылки на социальные сети после модерации), медиа (фото/видео), тип объявления (продажа, разведение, выставка, усыновление, stud_service), статус (активный, неактивный, на модерации, отклонен, архивирован)
- Гео-поиск: поиск по радиусу (1-100 км) от местоположения пользователя
- Фильтрация: по виду, породе, возрасту, полу, цене, местоположению, типу объявления
- Рабочий процесс модерации: пре-модерация (объявление появляется только после одобрения модератором)
- Действия пользователей: создание, редактирование, удаление собственных объявлений, избранные, compartir
- Интеграция с доменом животного (связывание объявления с сущностью животного)
- Интеграция с доменом идентификации (информация о продавце/владельце)
- Соответствие российским законам о рекламе и 152-ФЗ для персональных данных в объявлениях

**Исключает:**
- Системы аукционов/торгов - отложено
- Обработка платежей (внутриприложение транзакции) - отложено на этап 2
- Логистика доставки/отправки - отложено
- Оценки/отзывы пользователей для продавцов - отложено
- Продвигаемые объявления/реклама - отложено

## Ограничения
- **Юридическое:** Должно соответствовать Федеральному закону РФ "О рекламе", ветеринарным правилам торговли домашними животными и 152-ФЗ для обработки персональных данных (номера телефонов, имена в объявлениях).
- **Безопасность:** Предотвращение спама, мошенничества и вредоносного контента в объявлениях. Ограничение частоты запросов на создание объявлений.
- **Производительность:** Поиск объявлений с гео-радиусом < 2с при нагрузке (цель <1с).
- **Масштабируемость:** Поддержка 100k+ активных объявлений.
- **Технологии:** Должно соответствовать выбранному стеку (NestJS, TypeScript, PostgreSQL).
- **Удобство использования:** Простой поток создания объявлений для массовых пользователей; четкое отображение основной информации.

## Предыдущие решения
- Сущность объявления включает: id, название, описание, цена, местоположение (широта/долгота), тип объявления (enum), статус (enum), продавецId (ВНЕШНИЙ КЛЮЧ к Пользователю), animalId (ВНЕШНИЙ КЛЮЧ к Животному, опционально), createdAt, updatedAt, moderatedAt, rejectedReason (опционально), viewsCount.
- Местоположение хранится как точка PostGIS (или как двойные широта/долгота с индексацией) для гео-поиска.
- Медиа: хранить URL в объектном хранилище (S3) для изображений/видео.
- Модерация: объявления имеют статус: черновик, ожидает, активно, отклонено, архивировано. Только активно объявления доступны для публичного поиска.
- Гео-поиск: использование PostgreSQL с расширением PostGIS или куб lattice для запросов по радиусу.
- Цена может быть null (бесплатное усыновление) или числовой.
- Контактная информация: после модерации отображать номер телефона и ссылки на социальные сети (Telegram, VK), предоставленные продавцом.

## Критерии верификации
- [ ] Модульные тесты >90% покрытия для модуля pet-marketplace (backend)
- [ ] Интеграционные тесты покрывают: создание объявления (валидное/невалидное), поиск по гео-радиусу, фильтрация, рабочий процесс модерации, переходы статусов
- [ ] End-to-end тесты покрывают: пользователь создает объявление, ищет объявление, просматривает детали объявления, редактирует объявление
- [ ] Ручное тестирование: проверка точности гео-поиска, рабочий процесс модерации, загрузка изображений
- [ ] Производительность: гео-поиск с 100k объявлениями возвращает результат за <1с для 95% запросов
- [ ] Соответствие: объявления не раскрывают персональные данные сверх разрешенного (телефон/социальные сети после модерации); соблюдение российских законов о рекламе
- [ ] Документация: спецификация OpenAPI сгенерирована и доступна
