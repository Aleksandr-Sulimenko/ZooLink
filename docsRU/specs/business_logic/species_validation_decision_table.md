# Таблица решений для валидации видов животных

## Обзор
Эта таблица решений определяет правила валидации для объявлений о животных на основе комбинаций вида/породы. Каждое правило указывает обязательные поля, ограничения и логику валидации.

## Структура таблицы решений

Действия (A1-AN) представляют требования валидации.
Столбцы R (Правило) представляют конкретные комбинации.

## Легенда
* **Опасные породы** (для собак): Американский питбультерьер, Стаффордширский терьер, Ротвейлер, Доберман и т.д., как определено местным законодательством
* **Юрисдикция**: РФ = Российская Федерация, ЕС = Европейский Союз, США = Соединенные Штаты, ЛЮБАЯ = применимо ко всем юрисдикциям
* **Единица измерения возраста**: Указывает, измеряется ли возраст в месяцах, годах или неделях
* **Требуемые документы**: Документы, которые должны быть предоставлены/загружены вместе с объявлением
* **Специальные ограничения**: Дополнительные правила валидации, выходящие за рамки стандартной документации

## Правила по умолчанию (если нет совпадения)
- **Вид не указан в списке**: Требуется микрочип или татуировка + базовый ветеринарный сертификат
- * **Возраст ниже минимального**: Объявление отклоняется с ошибкой "Животное слишком молодо для передачи/продажи"
- * **Отсутствуют обязательные документы**: Объявление заблокировано до предоставления документов
- * **Недопустимый формат ID**: Специфичное сообщение об ошибке в зависимости от типа ID (например, "Недопустимый формат клейма")

## Примечания по реализации
1. Эта таблица решений должна быть реализована как механизм правил, который оценивает условия по порядку
2. Первое совпавшее правило (сверху вниз) определяет требования валидации
3. Все валидации должны пройти, прежде чем объявление может перейти из состояния ЧЕРНОВИК в состояние НА МОДЕРАЦИИ
4. Ошибки валидации должны возвращаться как ошибки конкретных полей с понятной пользователем подсказкой
5. Юрисдикция должна определяться из расположения объявления (страна/регион)
6. Определение категории породы может потребовать поиска во внешней базе данных пород для сложных случаев
