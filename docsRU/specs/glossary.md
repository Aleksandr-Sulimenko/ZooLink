# Глоссарий терминов ZooLink

## Основные сущности

**Пользователь** — лицо, зарегистрированное в системе ZooLink, которое может выполнять действия в соответствии со своей ролью (владелец домашнего животного, заводчик, фермер, модератор, администратор).

**Объявление** — запись в системе, размещенная пользователем для продажи, разведения, усыновления или выставки животного. Содержит информацию о животном, местоположении, цене, типе объявления и контактных данных после модерации.

**Животное** — сущность, представляющая домашнее животное или скот, с атрибутами вид, порода, возраст, здоровье, идентификационные данные (микрочип, бирка для ушей) и ссылкой на владельца.

**Гео‑поиск** — функционал поиска объявлений и животных в указанном радиусе (1-100 км) от местоположения пользователя с использованием геокоординат.

## Нормативные акты

**152-ФЗ** — Федеральный закон Российской Федерации «О персональных данных», регулирующий сбор, хранение, использование и защиту персональных данных.

**149-ФЗ** — Федеральный закон Российской Федерации «Об информации, информационных технологиях и о защите информации», регулирующий модерацию информации и защиту информации.

## Технические термины

**C4 Model** — методика визуализации архитектуры программного обеспечения на четырех уровнях: контекст, контейнеры, компоненты и код.

**STRIDE** — модель оценки угроз безопасности, включающая catégories: Spoofing, Tampering, Repudiation, Information disclosure, Denial of service, Elevation of privilege.

**PostGIS** — расширение PostgreSQL для хранения и обработки географических объектов.

**JWT** — JSON Web Token, стандарт для создания токенов доступа, которые утверждают определенные утверждения.

**RBAC** — Role-Based Access Control, модель контроля доступа basée на ролях пользователей.

**SMS** — Short Message Service, сервис передачи текстовых сообщений, используемый для аутентификации по телефону.

**OAuth** — открытый стандарт авторизации, позволяющий пользователям предоставлять сторонним приложениям доступ к своим ресурсам без передачи учетных данных.

**Ger** — нет, пропустим.

## Бизнес-процессы

**Модерация** — процесс проверки объявлений moderatorom перед публикацией, включающий проверку соответствия правилам и законодательству.

**Архивирование** — процесс перемещения объявлений или записей о животных в состояние архива вместо удаления для сохранения исторических данных и соответствия законодательству.

**Разведение** — процесс подбора животных для спаривания с целью получения потомства, включающий поиск партнеров по критериям вида, породы, здоровья и родословной.
